# JavaCard sample applet

This repository contains the minimal amount of setup to build JavaCard applications.

`build.xml` can be modified to add imports, to change the AID, to change the package and class names, etc.

The build system is configured to use JavaCard 3.0.4, but any version can be used within the requirements
of [ant-javacard](https://github.com/martinpaljak/ant-javacard).

Install the applet using [GlobalPlatformPro](https://github.com/martinpaljak/GlobalPlatformPro), simulate it
with [JCardEngine](https://github.com/martinpaljak/JCardEngine).
